# 🐦 Bird Species Observation Analysis

## 📌 Project Overview
This project analyses bird species distribution across Forest and Grassland ecosystems using real-world observational data. The goal is to understand biodiversity patterns, environmental impact, and species behaviour.

---

## 🎯 Objectives
- Analyse species distribution across habitats
- Study temporal patterns (monthly trends)
- Understand environmental impact (temperature, humidity, sky)
- Evaluate detection methods and distance-based observations
- Build an interactive dashboard using Streamlit
- Develop a predictive model for species occurrence

---

## 🛠️ Tech Stack
- Python (Pandas, NumPy)
- Plotly (Visualisation)
- Streamlit (Dashboard)
- Scikit-learn (Machine Learning)

---

## 📊 Key Features
- Interactive dashboard with filters
- Species distribution analysis
- Monthly activity trends
- Environmental impact analysis
- Detection behaviour insights
- Hotspot identification
- ML-based species prediction

## 📈 Insights
- Grasslands show higher observation density
- Forests support unique species diversity
- Bird activity peaks during moderate temperatures
- Detection bias exists based on distance
- Environmental factors influence behaviour
